# Privacy Policy

Last Updated: 15 June 2026

Your privacy is important. This Privacy Policy explains what data DEATH Security Bot may collect and how it is used.

1. **Data We May Collect**

DEATH may collect and store limited Discord-related data required for bot features to work properly, including:

Server IDs User IDs Channel IDs Role IDs Command usage data Server configuration settings Moderation logs Automod settings Anti-nuke settings Warning, mute, ban, or punishment records Welcome and logging channel settings

This data is used only to provide bot functionality.

2. **Data We Do Not Collect**

DEATH does not collect:

Discord passwords Personal addresses Payment information Private messages unless a command or feature directly requires bot interaction Sensitive personal information outside Discord functionality

We do not sell user data.

3. **How Data Is Used**

Collected data may be used to:

Run moderation commands Protect servers from nukes and raids Save server settings Provide logs and security alerts Track warnings or punishments Improve bot stability and performance

4. **Data Sharing**

We do not sell, rent, or trade your data.

Data may only be shared if required by law, Discord policy enforcement, or to prevent abuse of the bot.

5. **Data Removal**

Server owners may request removal of stored server data by contacting the support server.

Support Server: [https://discord.gg/d4hz](https://discord.gg/d4hz)

Once removed, some features may stop working until the bot is configured again.

6. **Security**

We take reasonable steps to protect stored data, but no online service can be guaranteed to be fully secure.

By using <mark style="color:$danger;">DEATH</mark>, you understand and accept this risk.
