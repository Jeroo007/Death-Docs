# Logging System

Logging helps your staff see exactly what happened.

## Suggested channels

```bash
>logs set security #security-alerts
>logs set moderation #mod-logs
>logs set automod #automod-logs
>logs set joins #join-logs
```

## Recommended log types

- Member logs
- Moderation logs
- Voice logs
- Role logs
- Channel logs
- Security logs
- Webhook logs
