# First Setup

## Suggested setup order

```bash
# Step 1
>logs set #death-logs

# Step 2
>antinuke enable

# Step 3
>antiraid enable

# Step 4
>automod links on
>automod spam on

# Step 5
>ping
```

## Suggested log channels

- `death-logs`
- `mod-logs`
- `automod-logs`
- `security-alerts`

> Use a private staff-only category for logs.
