# Required Permissions

For best results, invite the bot with **Administrator** permissions.

## Important Permissions

| Permission | Why it matters |
|---|---|
| Manage Roles | Role control and protection |
| Manage Channels | Restore and protect channels |
| Ban Members | Stop raiders and nukers |
| Kick Members | Remove harmful users |
| Moderate Members | Timeout spam users |
| Manage Webhooks | Prevent webhook abuse |
| View Audit Log | Detect dangerous admin actions |
| Manage Messages | Delete spam and harmful content |

> The bot can only manage roles **below its own role position**.
