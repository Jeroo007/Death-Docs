# How to Import

## Method 1 — Best method (GitHub + GitBook)

1. Extract this ZIP file
2. Upload the whole folder to a GitHub repository
3. Open GitBook
4. Create a new Space
5. Choose **Import with Git Sync**
6. Connect your GitHub repository
7. Import the docs

## Method 2 — Manual

1. Extract the ZIP file
2. Create a GitBook space
3. Copy `README.md` into the homepage
4. Use `SUMMARY.md` to create the sidebar pages
5. Copy each file into its matching GitBook page

## Important

- Keep the `assets/` folder
- Upload the GIFs and images too
- Make sure image paths remain the same if you use Git Sync
