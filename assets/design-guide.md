# GitBook Design Guide

This pack is designed around what GitBook handles well.

## Best supported visual features

- GIFs
- Static images
- Dark mode
- Code blocks
- Tables
- Collapsible sections

## Best style for commands

Use **fenced code blocks** like this:

```bash
>ping
>serverinfo
>botinfo
```

This gives the closest clean result to a **macOS terminal box** without depending on unsupported heavy custom CSS.

## Brand colors

- Background: `#050505`
- Card background: `#0C0C0C`
- Accent red: `#8B0000`
- Text: `#EAEAEA`
